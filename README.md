# FroggerGame
A lightweight recreation of the classic Frogger arcade game built with C and OpenGL

## Download
[**Download Latest Release (Windows x64)**](https://github.com/vonrimez/FroggerGame/releases/latest)

### Features
Local and custom levels, with the ability to create your own levels.

### Controls
WASD, Arrow keys, and some secret buttons (N - noclip, H - show hitboxes, L - increase the level).

### Main Goal
Fill all the houses while avoiding cars, snakes, and lawnmowers. Stay out of the water by jumping on logs and other frogs.
